# stocktracking


This application is a financial asset monitoring dashboard, particularly suited to US equities. You will find technical indicators such as MACD, RSI, ADX ... As well as statistical techniques such as time series analysis with an additive decomposition model. The use is rather intuitive; you enter the tickers of the assets you are interested in, separated by a space (in lowercase or uppercase, it doesn't matter) in the input tab, and you can find your interactive charts in the output part.
Two things to know however: 
- My application uses the yahoo API, so to avoid getting the tickers wrong, check their names on yahoo.
- Also, for everything to work properly, make sure that the asset / assets you want to follow are not too recent on the market. Ideally, there should be at least 5 years of data available.

To start using stocktracking, click on this link: https://stocktracking.herokuapp.com/

Enjoy
